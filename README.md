# ☕ Sistema de Pedidos de Cafeteria

Projeto prático desenvolvido para a disciplina de **Verificação e Validação de Software**, focado na aplicação de boas práticas de teste e desenvolvimento.

### Conceito
Um sistema interno de pedidos para cafeteria, operado diretamente por um atendente no balcão. A aplicação tem como foco principal o cálculo preciso de preços, aplicação de cupons, validação rigorosa de regras de negócio e uma suíte robusta de testes automatizados.

### Stack
* **Backend:** FastAPI (Python)
* **Banco de Dados:** SQLite com SQLAlchemy 2.0
